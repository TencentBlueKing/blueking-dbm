## mongod
```json
{
  "ip":"10.1.1.1",
  "port":27002,
  "sourceIP":"10.1.1.3",
  "sourcePort":27007,
  "sourceDown":true,
  "adminUsername":"xxx",
  "adminPassword":"xxxxxxxx",
  "targetIP":"10.1.1.1",
  "targetPort":27004,
  "targetPriority":"",
  "targetHidden":""
}
```
"sourceDown" 源端是否已down机
"targetPriority"可以指定替换节点的优先级
"targetHidden"可以指定替换节点是否为隐藏节点
