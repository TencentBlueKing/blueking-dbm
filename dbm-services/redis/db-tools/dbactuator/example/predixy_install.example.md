# predixy_install
## 安装predixy
`./dbactuator_redis  
--uid={{uid}}
--root_id={{root_id}}
--node_id={{node_id}}
--version_id={{version_id}}
--atom-job-list="predixy_install"
--payload='{{payload_base64}}'`

## 前置工作:
- 先运行 `./dbactuator_redis --atom-job-list="sysinit"`
- 将`predixy-1.0.5.tar.gz`下载到`/data/install`目录下;
